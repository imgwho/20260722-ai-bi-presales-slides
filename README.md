## 📊 作品集核心内容概述

1. **AI + BI 开源项目**
   - **cwtwb**: 让 AI 工程化构建 Tableau 工作簿（生成 `.twb / .twbx`）。
   - **cwprep**: 声明式构建 Tableau Prep 数据清洗流程（生成 `.tfl / .tflx`）。
   - **验证**: 社区累计 30,000+ 下载量，获得 Tableau 官方生态认可。

2. **AI 产品 - 销售话术分析系统**
   - 从 4.4GB+ 企业微信聊天数据中筛选样本，结合 DuckDB + DeepSeek 进行多阶段分析与自动评测，生成可评分的话术与聊点库。

3. **华芝林 · 个人售前项目闭环**
   - AI 原生需求澄清与冲突识别，将复杂的返款业务拆解为“应返”与“实返”双可追溯工作流，并通过可点击原型锁定一期交付边界。

4. **多行业案例矩阵**
   - **制造**: 一汽 / 一汽大众（产销存、供应链、财务经营）
   - **消费**: 立白 / 蒙牛 / 华芝林（电商全链路、销售与返款数智化）
   - **保险**: 友邦 / 安盛天平（业绩、佣金、保费异常监控）
   - **售前推进**: 金科 / 广汽研究院（招投标、产品边界、平台升级）

## 🎨 视觉主题说明

项目采用了源自 [`zarazhangrui/beautiful-html-templates`](https://github.com/zarazhangrui/beautiful-html-templates) 的 **Bold Poster** 设计语言：

- **字体搭配**: `Shrikhand` (特大Display标题) + `Libre Baskerville` (典雅正文) + `Space Grotesk` (强排版标签/数字)
- **色彩与质感**: 黑红极简高对比色调 (`#1C1410`, `#D8000F`, `#FFFFFF`)，搭配硬朗 `4px` 偏移投影与红色底层进度条。
