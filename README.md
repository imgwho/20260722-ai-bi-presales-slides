# 郭文华｜AI + BI 业务型售前案例网站

基于 reveal 风格网页演示制作的个人面试作品集，内容包括：

- cwtwb / cwprep AI+BI 开源项目
- 销售话术分析与聊点生成系统
- 华芝林个人售前项目：AI 辅助需求澄清、流程建模、原型与范围控制
- 制造、消费、保险等行业案例矩阵

访问根目录会自动进入 `revealjs/` 演示版本。支持键盘左右键、PageUp/PageDown、空格以及页面按钮翻页。

---

## 本地预览

直接打开根目录的 `index.html` 或 `revealjs/index.html`，也可以使用 npm 脚本启动预览：

```powershell
npm run reveal
```
