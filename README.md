## 🚀 项目核心亮点

1. **三套幻灯片系统驱动**：
   - `markdown/`：采用标准的 Markdown (`slides.md`) 语法，支持 Markdown 内容实时读取、自动化卡片转换与 HTML 混合渲染；
   - `bold-poster/`：现代高端商业海报风格，支持大字号高对比卡片与全屏高亮；
   - `revealjs/`：经典全屏过渡与交互模式。

2. **核心业务与开源项目展示**：
   - **定位**：8年+ 经验、30+ 售前经历、8大行业覆盖、3个开源项目、端到端售前 5 大数字标签；
   - **开源生态**：`cwtwb` (Tableau 自动构建)、`cwprep` (Tableau Prep 自动化清洗)、`tableauPushDing` (BI 报表长图拼接与钉钉自动推送)；
   - **AI 产品**：销售话术分析系统（数据流架构、白酒专家背书高价值案例、SEED+P 真实对话上下文与 8 大类高频聊点全景库）；
   - **售前项目**：华芝林医药返款数智化（混乱需求理清、应返/实返双工作流、5大系统交付规范、核销工作台原型与全国大盘）。

## 💻 本地运行说明

安装或使用任意静态服务器运行项目根目录：

```bash
# 启动本地开发服务
npx http-server . -p 8080
```

- 访问 Markdown 驱动版：`http://localhost:8080/markdown/`
- 访问 Bold Poster 版：`http://localhost:8080/bold-poster/`
- 访问 Reveal 版：`http://localhost:8080/revealjs/`
